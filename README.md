# BMI Calculator

A simple Python command-line application that calculates the Body Mass Index (BMI) of a user.

## Features

- Accepts weight (kg) and height (m)
- Calculates BMI using the standard formula
- Classifies BMI into:
  - Underweight
  - Normal
  - Overweight
  - Obese
- Rounds BMI to 2 decimal places
- Validates invalid and negative inputs

## Technologies Used

- Python 3

## How to Run

1. Open the project in Visual Studio Code.
2. Open the terminal.
3. Run:

```bash
python bmi_calculator.py
```

## Example

```
Enter your weight (kg): 70
Enter your height (m): 1.75

Your BMI is: 22.86
Category: Normal
```
